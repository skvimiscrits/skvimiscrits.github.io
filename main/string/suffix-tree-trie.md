## Suffix trie

Suffix trie is a specific case of a trie which is formed by all suffixes
of a string.

A suffix trie can be formed for a length $n$ string in $O(n^2)$.

A typical use case is matching a pattern $p$ of length $m$ in $O(m)$
time.
